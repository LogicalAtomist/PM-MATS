NodeVisualizer

Class to visualize a Node object in 2D space. It can calculate the position of a node based on its level, and draw it on a given SVG container.

new NodeVisualizer(node: Object, x: number, y: number)
Parameters
node (Object) The Node object to visualize.
x (number = 0) The initial x-coordinate.
y (number = 0) The initial y-coordinate.
Instance Members
▸ getLevel()

Get the level of the node based on the decimal part of its number property.

getLevel(): number
Returns
number: The level of the node.
▸ getPosition()

Calculate the position [x, y] of the node based on its level.

  • Level 0 nodes remain at their initial position.
  • Odd level nodes are positioned vertically.
  • Even level nodes are positioned horizontally.
getPosition(): Array<number, number>
Returns
Array<number, number>: An array containing the new x and y coordinates.
▸ draw(containerId, color?)

Draw the node in the given SVG container.

  • Creates a circle at the node's position.
  • Adds a number label above the circle.
  • Sets up a click event listener on the circle.
draw(containerId: string, color: string?)
Parameters
containerId (string) The id of the SVG container where the node will be drawn.
color (string?) The color to fill the circle. Defaults to 'red'.
