▸ getChapterNodes(chapter)

Fetches the nodes associated with a specific chapter, ensuring that there's a node for every primary mantissa value from 0 to 9. Missing nodes are represented as placeholders.

getChapterNodes(chapter: (string | number)): Array<Object>
Parameters
chapter ((string | number)) The chapter for which the nodes are to be fetched.
Properties
type (string) : The type of the node. In this context, it's always "node".
properties (Object) : The properties associated with the node.
  • properties.number string

    The node's unique identifier in the format "{chapter}.{mantissa}".

  • properties.isPlaceholder boolean?

    Indicates if the node is a placeholder for a missing primary mantissa value.

Returns
Array<Object>: An array of node objects associated with the chapter, sorted by their 'number' property.
Example
// Assuming the instance has a nodes property populated as:
// {
//     node1: { properties: { chapter: '1', number: '1.0' } },
//     node2: { properties: { chapter: '1', number: '1.1' } },
//     ... other nodes
// }
getChapterNodes(1);
// Output: Array of nodes for chapter 1, with placeholders for missing primary mantissa values.
▸ plot(chapter, startingX, startingY)

Plots nodes for a specified chapter on a 2D space. Nodes are positioned based on their 'number' property's mantissa length.

Nodes with:

  • A mantissa length of 0 (like x.0) are treated as primary nodes.
  • A mantissa length of 1 are vertically aligned with primary nodes.
  • A mantissa length of 2 are horizontally displaced from the previous node.
  • A mantissa length of 3 are vertically aligned below their predecessor.

Special cases (e.g., the node with the number 25.1011) are handled uniquely.

plot(chapter: (string | number), startingX: number, startingY: number): [Array<Object>, number]
Parameters
chapter ((string | number)) The chapter for which the nodes are to be plotted.
startingX (number = 0) The initial x-coordinate for plotting.
startingY (number = 0) The initial y-coordinate for plotting.
Properties
x (number) : The x-coordinate of a node.
y (number) : The y-coordinate of a node.
Returns
[Array<Object>, number]: A tuple where the first element is the array of plotted node objects and the second is the maximum x-coordinate value among the nodes.
Example
// Given nodes with properties {chapter: '1', number: '1.x'} etc.
plot(1, 0, 0);
// Output: Array of plotted nodes for chapter 1 and the maximum x-coordinate value.
